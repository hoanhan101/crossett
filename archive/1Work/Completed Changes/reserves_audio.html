Guidelines for Putting Audio MP3 Files on Reserve

  1. If you would like to create the digital copies yourself, please refer to the instructions for how and where to create these files. If you would like Nino Mendolia Mendolia@bennington.edu to create the digital copies please provide the CDs, a typed list of what you would like digitized, and the course name.

  2. You will be notified when your material is posted. You will be sent a course password for this class; for copyright purposes, it is essential that you share this password with students enrolled in the course only. They will need this password each time they access the material.

  3. Please review your course reserves to be sure everything has been included. Notify Kathy KathyW@bennington.edu x4605 if anything is missing.

  4. Please read this important copyright information.
