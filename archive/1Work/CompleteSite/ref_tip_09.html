Reference Tip:

Remember variant spellings to collect all the fruits of your labour.

Some databases take care of the problem for you, so a search for {u201C}theater{u201D} automatically searches {u201C}theatre{u201D} as well. Do a search using each variant and compare the two result lists. If the number of results is the same for both, the database is searching both spellings. If the number of results differs, you need to ensure your searches are picking up both spellings. You can do this by typing the search both ways, or by utilizing {u201C}OR{u201D} in the search. You could also refer to the help section of the database to see if you can use truncations.
